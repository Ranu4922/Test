# Hex Survival Prototype (Godot 4)

## Projektstruktur

- `scenes/` enthält alle Godot-Szenen
- `scripts/` enthält GDScript nach Domain sortiert
- `data/` enthält balancierbare Spielwerte als JSON
- `assets/` enthält Platzhalter für Art/Audio
- `tests/` enthält spätere Testskripte

## Erste spielbare Features

1. Bau-Modus mit Taste `B`
2. Gebäudevorschau auf Hex-Feldern
3. Platzierungsregeln pro Terrain
4. Ressourcenkosten beim Platzieren
